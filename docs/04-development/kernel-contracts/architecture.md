# Kernel Contracts Architecture

The contracts layer defines the stable language of the AIStack Knowledge Operating System.

It contains conceptual interfaces for:

- Knowledge Artifacts
- Knowledge Providers
- Knowledge Engines
- Knowledge Generators
- Knowledge Renderers
- Knowledge Policies
- Context Bundles

Implementations depend on contracts.

Contracts do not depend on implementations.
