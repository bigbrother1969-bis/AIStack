# Kernel Contracts Specification

## Functional Requirements

- Define lightweight Python contracts.
- Avoid external dependencies.
- Support import from `aistack.contracts`.
- Keep the Kernel independent from specific providers or engines.

## Acceptance Criteria

- Contract modules compile.
- Contracts can be imported from `aistack.contracts`.
- No external dependency is required.
