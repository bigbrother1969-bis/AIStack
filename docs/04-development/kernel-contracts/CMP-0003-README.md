# CMP-0003 — Kernel Contracts

## Purpose

Define the stable conceptual contracts of the AIStack Kernel.

## Responsibilities

- Define the common language of the Knowledge Operating System.
- Keep implementations independent from technologies.
- Ensure components depend on contracts, not on concrete implementations.

## Principle

Contracts come before implementations.
